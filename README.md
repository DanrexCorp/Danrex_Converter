# Danrex Converter

Универсальный конвертер файлов. Входит в состав **Danrex Tools** — софт-направления **Danrex Corp**.

---

## Возможности

Конвертация между 300+ форматами файлов в одном приложении.

**Категории:**
- 🖼️ Изображения (50+ форматов)
- 🎬 Видео (30+ форматов)
- 🎵 Аудио (35+ форматов)
- 📄 Документы (50+ форматов)
- 🗜️ Архивы (40+ форматов)
- 🧊 3D Модели (40+ форматов)
- 🔤 Шрифты (15+ форматов)
- 📊 Таблицы и БД (30+ форматов)
- 💬 Субтитры (15+ форматов)
- 🎮 Игровые ресурсы (25+ форматов)

---

## Как использовать

1. Скачайте **Danrex Converter.exe**
2. Запустите
3. Выберите категорию (вкладка слева)
4. Выберите папку или файлы
5. Выберите целевой формат
6. Настройте параметры (качество, размер, битрейт)
7. Нажмите «Конвертировать»

---

## Требования

- Windows 7 / 8 / 10 / 11
- Для видео и аудио: требуется FFmpeg (если не найден — утилита покажет инструкцию)
- Ничего устанавливать не нужно
